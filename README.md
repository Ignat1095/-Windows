📖 Инструкция по применению
Настройте желаемые цвета с помощью визуального редактора выше
Скопируйте код или скачайте .reg файл
Если скопировали код: создайте текстовый файл, вставьте код и сохраните с расширением .reg
Запустите .reg файл двойным кликом
Подтвердите добавление данных в реестр
Перезагрузите компьютер для применения изменений
