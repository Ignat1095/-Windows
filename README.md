# Color Highlight Editor for Windows

Простое приложение для настройки цвета выделения текста и подсветки элементов в Windows через визуальный редактор.

## Как это работает

1. Приложение меняет значения в реестре по ключу:
   - `Hilight` – цвет выделения текста.
   - `HotTrackingColor` – цвет элементов при наведении.

2. Цвета задаются в формате RGB (0–255).

## Как применить

1. Запустите приложение.
2. Выберите нужный цвет:
   - Визуальный пикер или вручную через RGB‑поля.
3. Нажмите кнопку «Generate REG File».
4. Сохраните сгенерированный `.reg`‑файл.
5. Дважды кликните по файлу и подтвердите изменение реестра.
6. Перезапустите Windows или перезапустите приложение/проводник, чтобы цвета применились.
