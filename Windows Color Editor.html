🎨 Редактор цветов Windows

Визуальная настройка цветов выделения и отслеживания

Цвет выделения (Hilight)

RGB: 0, 120, 215
Цвет фона выделенного текста в Windows (например, в Проводнике или текстовых полях)

Цвет отслеживания (HotTracking)

RGB: 0, 102, 204
Цвет элементов при наведении курсора (ссылки, кнопки в интерфейсе Windows)

📋 Код для редактора реестра

Windows Registry Editor Version 5.00 [HKEY_CURRENT_USER\Control Panel\Colors] "Hilight"="0 120 215" [HKEY_CURRENT_USER\Control Panel\Colors] "HotTrackingColor"="0 102 204"

📖 Инструкция по применению

  1. Настройте желаемые цвета с помощью визуального редактора выше
  2. Скопируйте код или скачайте .reg файл
  3. Если скопировали код: создайте текстовый файл, вставьте код и сохраните с расширением .reg
  4. Запустите .reg файл двойным кликом
  5. Подтвердите добавление данных в реестр
  6. Перезагрузите компьютер для применения изменений
